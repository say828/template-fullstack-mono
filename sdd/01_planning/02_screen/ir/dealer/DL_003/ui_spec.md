# DL_003 UI Spec

- screen code: `DL_003`
- screen name: `매물보기`
- primary route: `/dealer/market`
- access: `role:DEALER`
- source spec: `sdd/01_planning/02_screen/ir/dealer/dealer_screen_spec.pdf`
- page number: `4`

## Screen Requirements

### 1. 메인 메뉴 영역
  - 현재 화면이 딜러의 매물 탐색 기본 진입점임을 고정 표
  - 시하는 주 메뉴 영역
  - • 매물보기 / 나의 입찰 / 거래·결제 / 설정 / 고객센터로
  - 업무 흐름 분기 제공
  - 나의 입찰 메뉴 클릭 시 (DL_010) 화면으로 이동
  - 거래/결제 메뉴 클릭 시 (DL_017) 화면으로 이동
  - 설정 메뉴 클릭 시 (DL_030) 화면으로 이동
  - 고객센터 메뉴 클릭 시 (DL_035) 화면으로 이동

### 2. 알림 아이콘
  - 딜러 계정에 도착한 거래·입찰·정산 관련 이벤트 알림 진
  - 입 트리거
  - 미열람 알림 존재 시 배지 표시, 클릭 시 알림 패널
  - (DL_001)오픈

### 3. 프로필 영역
  - 현재 로그인된 딜러 계정의 세션 제어 및 로그아웃 진입
  - 지점
  - 클릭 시 프로필 패널 (DL_002)오픈

### 4. 매물 필터 및 검색 영역
  - 전체 / 국내거래 / 수출가능 거래 유형 분기 필터 제공
  - 입찰중/ 입찰마감임박 상태 기준 필터 제공
  - 최신 등록순, 인기순, 최고 입찰순, 최저입찰순 정렬 기준
  - 제공
  - 차량명·차량번호 기반 키워드 검색 가능
  - 모든 필터·검색 변경 시 매물 리스트 즉시 재집계

### 5. 매물 카드 리스트 영역
  - 판매자가 등록한 차량 매물의 실시간 집계 노출 영역
  - 무한 스크롤 방식으로 연속 로드
  - 각 카드에 입찰 상태, 거래 유형, 차량 핵심 정보, 현재 최
  - 고 입찰가, 입찰 건수, 마감 잔여 시간 집계 노출
  - 입찰 가능 여부 및 마감 임박 상태 시각 강조
  - 마감된 매물의 경우 미노출 처리

### 6. 매물 상세 진입
  - 선택 매물의 상세 정보 확인 화면으로 이동
  - 클릭 시 해당 매물의 입찰 상태에 따른 매물상세(DL_004
  - 등) 화면으로 이동

### 7. 입찰 참여
  - 해당 매물에 대한 딜러 입찰 절차 진입 트리거
  - 클릭 시 입찰참여(DL_009) 패널 호출
  - 입찰완료한 매물의 경우, 해당 버튼명은 [입찰 완료]로 변
  - 경되며 클릭 불가능
