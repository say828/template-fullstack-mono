# DL_026 UI Spec

- screen code: `DL_026`
- screen name: `거래결제상세(인도진행)`
- primary route: `/dealer/transactions/:vehicleId/detail/delivery`
- access: `role:DEALER`
- source spec: `sdd/01_planning/02_screen/ir/dealer/dealer_screen_spec.pdf`
- page number: `27`

## Screen Requirements

### 1. 감가 협의 결과 영역
  - 판매자가 감가 제안을 승인하여 최종 감가 금액이 확정된
  - 거래 요약 영역
  - 감가 확정 금액: 판매자 승인 기준 최종 감가 합계 금액
  - 최종 거래가: 낙찰가 − 확정 감가 금액
  - 해당 값들은 이후 인도·송금·정산의 기준 금액으로 고정
  - 되며 변경 불가
  - [감가 협의 내역보기] 버튼 클릭 시 (DL_027) 화면 호출

### 2. 인도 일정 등록 영역
  - 딜러가 판매자와 1차 통화 후 확정한 인도 일정을 등록하
  - 는 입력 영역
  - 입력 항목: 인도일자, 인도 시간, 인도 장소, 판매자 연락
  - 확인
  - 판매자 연락처 : 판매자 회원 정보에 등록된 대표 연락처
  - 인도 일자/시간 : 실제 차량·서류 인수 예정 시점
  - 차량 인도 장소 : 차량 실물 인수 장소
  - 인도 일정 미등록 상태에서는 하위 영역(인도 진행 상태/
  - 체크리스트/완료 버튼) 전부 비활성

### 3. 인도 일정 등록 버튼
  - 입력된 인도 일정을 거래에 확정 반영
  - 등록 즉시 거래 단계는 인도 진행 상태로 전환
  - 등록 시 하위 영역(인도 진행상태/체크리스트) 활성화

### 4. 인도 진행 상세 영역
  - 인도 방식, 체크리스트, 필수 확인 항목 표시
  - 체크리스트 미완료 시 인도 완료 처리 불가
  - 사진 첨부
  - 차량 인도 사진
  - 서류 인수 사진
  - 파일 조건: PNG/JPG/PDF / 각 최대 5MB

### 5. 인도 완료 처리 버튼
  - 인도 과정이 모두 완료되었음을 확정하는 기능
  - 클릭 시 인도 완료 처리 및 거래 단계 송금/정산 단계로
  - 전환

### 6. 거래 요약 패널
  - 차량 정보, 낙찰가, 감가 확정 금액, 최종 거래가, 현재 거
  - 래 단계 표시
  - 모든 금액 및 단계 상태는 상단 진행 스텝과 실시간 동기
  - 화
- 인도 완료 처리가 되었습니다.
  - 확인
