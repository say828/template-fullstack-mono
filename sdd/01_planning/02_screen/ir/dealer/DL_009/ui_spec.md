# DL_009 UI Spec

- screen code: `DL_009`
- screen name: `입찰참여`
- primary route: `/dealer/market/:vehicleId/bid`
- access: `role:DEALER`
- source spec: `sdd/01_planning/02_screen/ir/dealer/dealer_screen_spec.pdf`
- page number: `10`

## Screen Requirements

### 1. 패널 닫기
  - 패널 닫기 아이콘 클릭 시 현재 입력 상태 초기화 후 패널
  - 닫힘

### 2. 매물 요약 영역
  - 입찰 대상 매물의 대표 이미지, 상태 태그, 차명, 연식, 주
  - 행거리, 연료, 변속기 요약 표시
  - ‘상세보기’ 클릭 시 매물 상세 화면(DL_004)으로 이동

### 3. 입찰 현황 정보 영역
  - 현재 최고 입찰가, 입찰 건수, 마감 시각, 잔여 시간 실시
  - 간 표시
  - 마감 시각 도달 시 본 패널 자동 비활성화

### 4. 내 입찰가 입력 영역
  - 딜러의 입찰 금액 입력 필드 제공
  - 숫자만 입력 가능, 천 단위 자동 콤마 적용
  - 사용자가 설정한 최소 입찰가 이하 금액 입력 시 오류 메
  - 시지 출력 및 입찰 불가
  - 마감 시각 이후 입력 불가 처리
  - +10만 / +50만 / +100만 / +500만 버튼 제공
  - • 현재 입력값 기준 해당 금액만큼 즉시 가산

### 5. 약관 동의 체크
  - 입찰 조건 및 거래 조건 확인 동의 필수 체크
  - 미체크 시 입찰 버튼 비활성화

### 6. 입찰하기 버튼
  - 유효 금액 + 약관 동의 충족 시에만 활성화
  - 클릭 시 서버로 입찰 요청 전송 및 입찰 이벤트 생성

### 7. 취소 버튼
  - 입력 내용 폐기 후 패널 닫힘
