# FRT_003 UI Spec

- screen code: `FRT_003`
- screen name: `로그인(딜러)`
- primary route: `/login?role=dealer`
- access: `public`
- source spec: `sdd/01_planning/02_screen/ir/seller/seller_screen_spec.pdf`
- page number: `4`

## Screen Requirements

### 1. 역할 선택 탭
  - 판매자 로그인 / 딜러 로그인 탭으로 구성됨
  - 현재 화면은 딜러 로그인 탭 선택 상태로 진입한 화면
  - 선택 상태는 로그인 시도 계정의 권한 기준값으로 적용됨
  - 선택된 역할과 계정 실제 역할이 불일치할 경우 로그인
  - 불가 처리됨
  - 판매자 로그인 탭 클릭 시 판매자 로그인(FRT_002) 화면
  - 으로 전환됨

### 2. 이메일 입력 필드
  - 딜러 계정 식별용 필수 입력 항목
  - 이메일 형식만 허용
  - 미입력 / 형식 오류 상태에서는 로그인 버튼 비활성화 유
  - 지
  - 로그인 실패 시에도 기존 입력값 유지됨

### 3. 비밀번호 입력 필드
  - 딜러 계정 인증용 필수 입력 항목
  - 입력값 마스킹 처리
  - 미입력 상태에서는 로그인 버튼 비활성화 유지

### 4. 자동 로그인
  - 체크 시 로그인 성공 후 자동 인증 상태 유지
  - 이후 접속 시 인증 절차 생략 가능

### 5. 로그인 버튼
  - 필수 입력값 충족 시에만 활성화
  - 클릭 시 딜러 계정 인증 시도
  - 인증 성공 시:
  - 딜러 권한 확정
  - 딜러 메인 홈 화면(DL_003)으로 전환
  - 인증 실패 시:
  - 오류 메시지 노출
  - 현재 화면 유지
  - 이메일 입력값 유지

### 6. 비밀번호 찾기
  - 클릭 시 비밀번호 찾기 화면(FRT_004)으로 이동
  - 비로그인 상태에서도 접근 가능

### 7. 회원가입
  - 클릭 시 회원가입 유형선택(FRT_005) 화면으로 이동
