# FRT_014 UI Spec

- screen code: `FRT_014`
- screen name: `상세보기(입찰중)`
- primary route: `/seller/vehicles/:vehicleId`
- access: `role:SELLER`
- source spec: `sdd/01_planning/02_screen/ir/seller/seller_screen_spec.pdf`
- page number: `15`

## Screen Requirements

### 1. 차량 정보 수정 버튼
  - 입찰 시작 전 상태일 경우에만 노출됨.
  - 입찰 기간 도래 및 입찰 진행 중 상태가 되면 본 버튼은 미
  - 노출됨.
  - 노출된 상태에서 클릭 시 차량 정보 수정 화면(FRT_033)
  - 으로 전환됨.
  - 차량 정보 수정 완료 후 본 화면으로 복귀됨.
  - 입찰이 시작된 이후에는 차량 정보 수정이 제한됨.

### 2. 차량 대표 영역
  - 차량 대표 이미지와 함께 현재 거래 상태 태그 ‘입찰중’,
  - 거래 유형 태그 ‘국내 거래’ 노출됨.
  - 해당 태그는 차량의 현재 거래 상태값 기준으로 고정 노
  - 출됨.

### 3. 입찰 진행 현황 영역
  - 현재 최고 입찰가, 입찰 건수, 입찰 마감일, 남은 기간, 진
  - 행 상태 바가 노출됨.
  - 표시 값은 실시간 입찰 데이터 기준으로 갱신됨.
  - ‘입찰 현황 보기’ 버튼 클릭 시 입찰 현황 상세 화면
  - (FRT_016) 으로 전환됨.

### 4. 차량 기본 정보 영역
  - 차량의 제조사, 모델, 연식, 주행 거리, 연료, 변속기, 색상,
  - 차량 번호가 노출됨.
  - 해당 정보는 판매자가 등록한 원본 데이터 기준으로 고정
  - 표시됨.

### 5. 차량 사진 및 옵션 영역
  - 등록된 차량 이미지가 슬라이드 형태로 노출됨.
  - ‘+5장 더보기’ 클릭 시 전체 이미지 갤러리 화면
  - (FRT_015)을 호출함.
  - 차량 주요 옵션 정보가 텍스트로 요약 노출됨.

### 6. 거래 진행 단계 영역
  - 거래 단계 타임라인이 노출됨.
  - 현재 단계 ‘입찰 진행 중’ 이 활성 상태로 표시됨.
  - 향후 단계는 비활성 상태로 예정 표시됨.

### 7. 최근 활동 영역
  - 해당 차량과 관련된 주요 이벤트 이력이 최신순으로 노출
  - 됨.
  - 예: 딜러 입찰 추가, 차량 정보 수정, 차량 등록 완료 등
  - 각 항목은 발생 일시와 함께 기록됨.
