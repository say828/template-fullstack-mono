# FRT_035 UI Spec

- screen code: `FRT_035`
- screen name: `거래정산내역`
- primary route: `/seller/settlement`
- access: `role:SELLER`
- source spec: `sdd/01_planning/02_screen/ir/seller/seller_screen_spec.pdf`
- page number: `36`

## Screen Requirements

### 1. 이번 달 정산 완료 건수
  - 선택된 조회 조건(5번) 중 기간 조건이 ‘이번 달’로 설정된
  - 상태를 기준으로, 정산 상태가 정산완료로 확정된 거래
  - 건수가 합산되어 노출됨.
  - 집계 대상은 거래 단계가 거래완료로 확정된 거래만 포함
  - 됨이며, 거래완료 이전 단계(입찰/검차/감가/인도·정산
  - 진행 중)는 집계에서 제외됨.
  - 5번 조건이 변경되는 즉시 1번 수치도 동일 기준으로 재
  - 산출되어 갱신됨.

### 2. 이번 달 정산 완료 금액
  - 1번과 동일한 집계 대상(거래완료 + 정산완료 확정) 기준
  - 으로 정산 금액(수령액)의 합계가 노출됨.
  - 정산 금액(수령액)은 각 거래의 최종 거래 금액에서 수수
  - 료가 차감된 결과값으로 확정된 값만 합산됨.
  - 정산 완료로 확정되지 않은 거래는 본 합계에 포함되지
  - 않음.

### 3. 이번 달 수수료 합계
  - 1번과 동일한 집계 대상 기준으로 수수료 금액 합계가 노
  - 출됨.
  - 수수료는 거래별 확정값으로 조회되며, 판매자가 본 화면
  - 에서 수정/재계산할 수 없음(조회 전용 고정값)

### 4. 정산 대기 건수
  - 거래 단계가 거래완료로 확정된 거래 중, 정산 상태가 정
  - 산대기로 유지되는 건수가 합산되어 노출됨
  - 정산대기는 정산 지급이 아직 확정되지 않은 상태로, 이
  - 후 어드민 확인 및 지급 완료 시점에 정산완료로 전환된
  - 값이 반영됨.
  - 본 수치는 판매자가 처리하는 값이 아니며, 외부 처리 결
  - 과가 반영될 때만 감소/증가됨.

### 5. 조회 조건 필터/검색 영역
  - 기간, 거래 유형, 정산 상태, 검색어 조건이 6번 리스트의
  - 조회 범위를 확정하는 기준 값으로 적용됨.
  - 기간: 시작일/종료일 직접 선택이 가능하며, 이번 달 / 지
  - 난 달 / 최근 3개월 프리셋 선택 시 기간 값이 즉시 해당
  - 범위로 치환되어 확정됨.
  - 거래 유형: 전체 기본값으로 노출되며, 선택 시 6번 리스
  - 트에 해당 유형만 포함되도록 조회 범위가 확정됨.
  - 정산 상태: 전체 기본값으로 노출되며, 정산대기/정산완
  - 료 선택 시 상태 기준으로만 리스트가 필터링됨.
  - 검색어: 차량명, 차량 번호 또는 거래번호 검색 입력값 기
  - 준으로 리스트가 제한됨. 공백 입력/미입력 상태에서는
  - 검색 조건 미적용으로 처리됨.
  - 조건이 변경되면 1~4의 요약 지표와 6번 리스트가 동일
  - 조건으로 동기 재조회되어 일관된 기준으로 맞춰짐.

### 6. 정산 내역 리스트
  - 조건(5번)으로 확정된 범위 내에서 거래 단위 정산 데이
  - 터가 행 단위로 노출됨.
  - 거래 완료일: 거래 단계가 거래완료로 확정된 일자(정산
  - 일과 별개)로 노출됨.
  - 차량 정보: 거래 대상 차량의 식별 정보(차량명 중심)로
  - 노출됨.
  - 거래 유형: 해당 거래의 유형 값(예: 국내 거래 / 해외(수
  - 출) 거래 등)으로 노출됨.
  - 최종 거래 금액: 감가 협의 결과로 확정된 거래 금액(거래
  - 기준 금액)으로 노출됨.
  - 수수료: 최종 거래 금액 기준으로 확정된 수수료 금액으
  - 로 노출됨.
  - 정산 금액(수령액): 판매자 수령 기준 확정 금액으로 노출
  - 됨(최종 거래 금액 - 수수료).
  - 정산일: 정산 상태가 정산완료로 확정된 경우에만 날짜가
  - 노출됨. 정산대기 상태에서는 -로 고정 노출됨.
  - 상태: 정산대기/정산완료 중 하나로 노출되며, 정산 완료
  - 는 되돌림 불가한 확정 상태로 유지됨.
  - 리스트 행 클릭 시 해당하는 거래의 상세 화면으로 이동
  - 함 (상태가 정산대기일경우 (FRT_037), 상태가 정산완료
  - 일경우 (FRT_036)화면 전환)
  - 리스트는 페이지네이션으로 제공되며, 페이지 변경 시에
  - 도 5번 조건은 유지된 상태로 조회 범위만 이동됨.
