# FRT_013 UI Spec

- screen code: `FRT_013`
- screen name: `내차량`
- primary route: `/seller/vehicles`
- access: `role:SELLER`
- source spec: `sdd/01_planning/02_screen/ir/seller/seller_screen_spec.pdf`
- page number: `14`

## Screen Requirements

### 1. 새 차량 등록하기 버튼
  - 우측 상단에 고정 노출됨.
  - 클릭 시 차량 등록 화면(FRT_033) 으로 전환됨.
  - 차량 등록 완료 후 본 화면 복귀 시 신규 차량 카드가 목록
  - 최상단에 추가됨.
  - 신규 차량의 거래 상태는 입찰대기로 확정되어 상태 태그
  - 가 즉시 반영됨

### 2. 거래 상태 필터 영역
  - ‘전체 / 입찰 중 / 입찰 마감 / 검차 / 감가 협의 / 인도·정
  - 산 / 거래 완료’ 탭 노출됨.
  - 선택 탭은 활성 컬러로 구분 노출됨.
  - 탭 선택 시 해당 상태에 해당하는 차량만 목록에 노출됨.
  - 탭 변경 시 목록은 초기화 후 재조회되며 스크롤 위치는
  - 최상단으로 이동됨.

### 3. 차량 카드 목록 영역
  - 판매자가 등록한 차량이 카드 형태로 무한 스크롤 방식으
  - 로 노출됨.
  - 각 카드에는 대표 이미지, 차량명, 연식, 주행거리, 연료,
  - 최고 입찰가, 입찰 건수, 거래 상태 태그, 상태 요약 정보
  - 가 고정 구조로 노출됨.
  - 거래 상태 태그는 상태값에 따라 컬러·텍스트가 분기 노
  - 출되며, 해당 값은 이후 이동할 상세 화면 분기의 단일 기
  - 준값으로 사용됨.

### 4. 상세보기 버튼
  - 모든 차량 카드에 공통 노출됨.
  - 클릭 시 해당 차량의 현재 거래 상태 기준 상세 화면으로
  - 전환됨.
  - 전환 대상 화면은 상태값 기준으로 자동 분기됨.
  - (FRT_014 등)

### 5. 입찰 현황 버튼
  - 차량 상태가 입찰중일 때만 노출됨.
  - 클릭 시 입찰 현황 상세 화면(FRT_016)으로 전환됨.
  - 입찰 진행 상태가 아닐 경우 미노출됨

### 6. 감가 내용 확인 버튼
  - 차량 상태가 감가 협의일 때만 노출됨.
  - 클릭 시 감가 협의 상세 화면(FRT_028)으로 전환됨.

### 7. 검차 일정 확인 버튼
  - 차량 상태가 검차일 때만 노출됨.
  - 검차 일정이 확정되지 않고, 검차 일정 요청이 들어왔을
  - 경우에 클릭 시 검차 일정 확인 화면(FRT_020)을 호출함
  - 검차 일정 요청이나 확정된 검차 일정이 없을 경우, 해당
  - 버튼 비활성화 처리
  - 검차 일정이 확정된 경우, 클릭 시 검차일정상세
  - (FRT_023) 화면을 호출함

### 8. 입찰자 선택하기 버튼
  - 차량 상태가 입찰 마감일 때만 노출됨.
  - 클릭 시 입찰자 선택 화면(FRT_018)으로 전환됨.

### 9. 정산 확인 버튼
  - 차량 상태가 인도·정산일 때만 노출됨.
  - 클릭 시 인도정산진행(FRT_030) 화면으로 전환됨.
