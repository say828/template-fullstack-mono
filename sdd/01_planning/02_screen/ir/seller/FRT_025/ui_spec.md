# FRT_025 UI Spec

- screen code: `FRT_025`
- screen name: `감가협의`
- primary route: `/seller/vehicles/:vehicleId/depreciation`
- access: `role:SELLER`
- source spec: `sdd/01_planning/02_screen/ir/seller/seller_screen_spec.pdf`
- page number: `26`

## Screen Requirements

### 1. 감가 협의 요약 카드 영역
  - 선택된 차량의 검차 완료 상태 및 감가 협의 단계 진입 상
  - 태가 상단 고정 영역으로 노출됨.
  - 차량 기본 정보(차량명, 연식, 주행거리, 연료)와 함께 현
  - 재 단계가 ‘감가 협의’로 확정 표기됨.

### 2. 차량 상세 이동 버튼
  - ‘차량 상세 보기’ 클릭 시 상세보기 화면(FRT_024)으로
  - 전환됨.
  - 이동 후에도 감가 협의 상태 유지되며, 상태 변경 없이 조
  - 회 전용으로만 진입 가능함

### 3. 딜러 감가 내역 영역
  - 딜러가 등록한 감가 항목 리스트가 노출되며, 감가 항목
  - 명, 사유, 감가 금액이 항목 단위로 노출됨.
  - 해당 데이터는 딜러 등록 + 평가 리포트 연동 기준 데이
  - 터로 고정되며, 판매자 수정 불가 및 서버 기준 정합 데이
  - 터로만 노출됨.
  - 총 감가 금액과 기존 낙찰가 대비 최종 거래 금액이 계산
  - 되어 노출됨.

### 4. 검차 요약 정보 영역
  - 검차일, 검차 장소, 담당 평가사 정보 노출됨.
  - 해당 데이터는 검차 일정 확정 시점 기준 고정 데이터이
  - 며 이후 변경 불가.

### 5. 검차 리포트 다운로드
  - PDF 형태의 검차 리포트 파일 다운로드 버튼 노출됨.
  - 다운로드 파일은 현재 감가 협의의 근거 자료로, 이후 분
  - 쟁 발생 시 기준 문서로 활용됨.

### 6. 분쟁/중재 안내 영역
  - 감가 내역에 대한 분쟁 발생 시 고객센터 중재 요청 가능
  - 상태 안내 문구 노출됨.
  - 클릭 시 1:1 문의(FRT_049) 화면으로 이동함

### 7. 재협의 요청 버튼
  - ‘재협의 요청하기’ 클릭 시 감가재협의 요청(FRT_026)
  - 화면을 호출함
  - 재협의 요청 시, 해당 감가 데이터는 임시 협의 상태로 전
  - 환되며 승인 이전까지 거래 금액은 확정되지 않음

### 8. 감가 금액 승인하기 버튼
  - 클릭 시 거래 금액은 최종 거래 금액으로 서버 확정 저장
  - 되며, 모든 이후 정산/송금/인도 프로세스는 해당 금액 기
  - 준으로 진행됨
  - 감가 금액 승인 후, 딜러가 인도 일정 등록 시 거래 상태는
  - 인도/정산 으로 전환됨
