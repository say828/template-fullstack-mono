# FRT_038 UI Spec

- screen name: `계정정보`
- primary route: `/settings/account`
- access: `role:SELLER`
- source spec: `sdd/01_planning/02_screen/ir/seller/seller_screen_spec.pdf`
- page number: `39`

## Screen Requirements

### 1. 설정 메뉴 / 좌측 네비게이션 영역
  - 상단 GNB에서 설정 진입 시 본 화면 기본 진입 위치로 고
  - 정됨.
  - 좌측 메뉴는 계정 정보가 선택 상태로 표시되며, 동일 영
  - 역에서 정산 계좌 / 알림 설정 / 보안·비밀번호 / 언어 및
  - 지역 / 약관 및 회원탈퇴 이동 경로 제공됨.
  - 본 화면에서 입력된 정보가 저장되지 않은 상태에서 타
  - 메뉴 이동 시 변경 데이터 유지 불가 상태로 처리됨(저장
  - 전 이탈 시 입력값 폐기).

### 2. 계정 기본 정보 영역
  - 판매자 계정의 식별 기준 정보 블록으로 구성됨.
  - 프로필 이미지, 계정 표시명, 로그인 이메일 정보가 함께
  - 노출됨.
  - 이름·이메일은 가입 시점 식별 정보로 고정, 본 화면에서
  - 수정 불가 상태 유지됨.

### 3. 연락처 정보 입력 영역
  - 판매자의 거래·정산·서류 발송·운영 연락 기준 데이터 입
  - 력 영역으로 구성됨.
  - 휴대폰 번호
  - • 판매자 공식 연락 수단 기준값으로 저장됨.
  - • 딜러·평가사·고객센터가 해당 번호를 기준으로 모든 거
  - 래 연락을 수행함.
  - • 입력 형식 불일치 시 저장 불가 상태로 전환됨.
  - 국가
  - • 단일 선택 드롭다운.
  - • 선택 국가에 따라 거주 지역 선택 후보가 재구성됨.
  - 거주 지역(광역 / 하위)
  - • 2단 선택 구조.
  - • 국가 변경 시 기존 선택값이 무효가 되는 경우 자동 초
  - 기화됨.
  - • 필수 입력 항목으로, 미선택 시 저장 불가 상태 유지
  - 본 영역의 정보는 이후 생성되는 모든 거래·정산·서류 처
  - 리의 기준 데이터로 즉시 반영됨.

### 4. 저장 처리 버튼
  - 저장 버튼 클릭 시 다음 조건을 모두 충족해야 저장 가능
  - 상태 유지됨. (휴대폰 번호 형식 유효 / 국가 선택 완료 /
  - 거주 지역(광역/하위) 선택 완료)
  - 조건 미충족 시 저장 차단 상태 유지, 오류 기준 필드 단위
  - 로 피드백 노출
  - 저장 완료 시 해당 시점부터 모든 거래·정산·서류 처리에
  - 신규 기준값 적용.
  - 저장 전까지는 기존 기준 데이터 유지됨.
