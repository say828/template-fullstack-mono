# FRT_006 UI Spec

- screen code: `FRT_006`
- screen name: `판매자회원가입`
- primary route: `/signup/seller`
- access: `public`
- source spec: `sdd/01_planning/02_screen/ir/seller/seller_screen_spec.pdf`
- page number: `7`

## Screen Requirements

### 1. 기본 정보 입력 영역
  - 해당 영역은 판매자 계정 생성에 필요한 필수 정보 입력
  - 블록으로 구성됨.
- 이름 :
  - 실명 기준 입력 필드
  - 미입력 시 가입 불가
- 이메일 주소 :
  - 로그인 계정 ID로 사용
  - 이메일 형식만 허용
  - 중복 가입 불가 (이미 가입된 이메일은 가입 차단)
- 비밀번호 :
  - 최소 보안 기준 충족 필수
  - 마스킹 처리
- 비밀번호 확인 :
  - 비밀번호 입력값과 일치 여부 검증
  - 불일치 시 가입 불가
- 휴대전화번호 :
  - 숫자만 입력 허용
  - 향후 정산 및 거래 알림의 기본 수신 채널로 사용
- 거주 국가 :
  - 국가 선택 드롭다운 제공
  - 선택 값은 거래 통화, 정산 정책, 서비스 노출 범위 판단에
  - 활용됨
- 약관 동의 :
  - 서비스 이용약관: 필수
  - 개인정보처리방침: 필수
  - 마케팅 정보 수신 동의: 선택
  - 필수 약관 미동의 시 가입 불가

### 2. 회원가입 완료 버튼
  - 상기 필수 정보 입력 및 필수 약관 동의 완료 시에만 활성
  - 화
  - 클릭 시:
  - 입력 정보 최종 검증
  - 판매자 계정 생성
  - 판매자 회원가입 완료 (FRT_007)화면으로 즉시 이동
  - 가입 완료 후 별도 승인 절차 없음

### 3. 소셜 회원가입
  - Google / Naver 회원가입 제공
  - 소셜 인증 완료 시:
  - 필수 정보 수집 단계로 이어짐
  - 약관 동의 절차 필수 진행
  - 가입 완료 후 판매자 권한 즉시 활성화

### 4. 로그인 전환
  - 기존 계정 사용자를 위한 로그인 전환 링크
  - 클릭 시 로그인 화면(FRT_002)으로 이동
  - 현재 입력된 정보는 저장되지 않음
