# FRT_028 UI Spec

- screen code: `FRT_028`
- screen name: `감가내용확인`
- primary route: `/seller/vehicles/:vehicleId/depreciation/completed`
- access: `role:SELLER`
- source spec: `sdd/01_planning/02_screen/ir/seller/seller_screen_spec.pdf`
- page number: `29`

## Screen Requirements

### 1. 감가 협의 완료 요약 영역
  - 차량 대표 이미지/차량명/연식/주행거리/연료 등 식별
  - 정보가 고정 노출됨.
  - 상태 태그는 ‘감가 ㄴ협의 완료’로 표시되어 감가 단계 종
  - 료가 시각적으로 확정됨.
  - 최초 낙찰가 및 검차 완료 등 선행 단계 결과가 요약 값으
  - 로 노출되어, 현재 화면이 “가격 확정 단계” 이후임을 인
  - 지 가능함.
  - 본 영역의 노출 값은 협의 완료 확정 시점의 서버 기준 스
  - 냅샷으로 고정되며, 이후 감가 항목 변경 또는 재산출 발
  - 생 없음.

### 2. 차량 상세 보기 버튼
  - 클릭 시 해당 차량의 차량 상세 화면(해당 상태 기반 상세
  - 화면)으로 이동됨.
  - 이동 후에도 거래 상태는 감가 협의 완료로 유지되며, 감
  - 가 협의 관련 조작(재협의 요청/승인 등)은 미노출
  - 본 버튼은 “상세 정보 확인 경로 제공” 목적이며, 감가 결
  - 과 자체를 변경하는 진입점으로 사용되지 않음.

### 3. 딜러 감가 내역 영역
  - 딜러가 제출한 감가 내역이 항목 단위(항목명/사유/감가
  - 금액)로 고정 노출됨.
  - 감가 금액은 마이너스(차감) 값으로 표기되어 최종 거래
  - 금액 산출 구조가 명확히 유지됨.
  - 하단에는 기존 낙찰가 / 감가 금액 소계 / 최종 거래 금액
  - 이 정산 기준 계산값으로 노출됨.
  - 최종 거래 금액은 본 거래의 결제/정산 기준 금액으로 확
  - 정되어, 이후 인도/정산 단계에서 동일 값이 참조됨.
  - 본 영역의 데이터는 감가 협의 완료 확정 시점 이후 수정/
  - 삭제/재계산 불가 상태로 고정됨.

### 4. 검차 요약 정보 영역
  - 검차일/검차 장소/담당 평가사 정보가 검차 확정 데이터
  - 기준으로 고정 노출됨.
  - “검차 리포트 다운로드(PDF)” 버튼을 통해 해당 거래에
  - 귀속된 검차 리포트 원본 파일을 조회 가능함.
  - 본 영역은 감가 내역의 근거 출처(검차 결과)가 거래에 귀
  - 속되어 있음을 확인시키는 검증 블록이며, 감가 완료 상
  - 태에서도 항상 조회 가능
