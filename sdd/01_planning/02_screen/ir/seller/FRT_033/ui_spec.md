# FRT_033 UI Spec

- screen code: `FRT_033`
- screen name: `차량등록`
- primary route: `/seller/vehicles/register`
- access: `role:SELLER`
- source spec: `sdd/01_planning/02_screen/ir/seller/seller_screen_spec.pdf`
- page number: `34`

## Screen Requirements

### 1. 차량 기본 정보
  - ‘거래 유형’은 국내 거래 / 수출 가능 2값 라디오로 구성되
  - 며, 선택값이 본 거래의 딜러 참여 범위 정책으로 확정됨.
  - 수출 가능 선택 시에만 이후 ‘딜러 노출 국가(수출 가능
  - 시)’ 입력이 노출/활성화됨
  - 국내 거래 선택 시 해당 입력은 미노출.
  - ‘연료’, ‘변속기’는 선택형 드롭다운으로 구성되며, 미선택
  - 상태에서는 placeholder가 유지됨.
  - 선택 완료 시 값이 즉시 반영되며, 이후 입찰 카드/상세
  - 요약에 고정 속성값으로 노출됨.
  - ‘제조사’, ‘연식’은 선택형 드롭다운으로 구성되며, 제조사
  - 선택 결과에 따라 ‘모델’ 입력의 유효성(허용 값 범위)이
  - 달라질 수 있음(정책상 자유 입력 유지 시에도 제조사-모
  - 델 조합이 검차/감가 판단에 사용됨).
  - ‘색상’, ‘모델’, ‘차량 번호’는 텍스트 입력으로 구성되며,
  - ‘차량 번호’는 형식 검증 대상 값으로 처리됨(형식 불일치
  - 시 다음 단계 진행이 제한됨).
  - ‘배기량(cc)’, ‘주행 거리(km)’는 숫자 입력으로 구성되며,
  - 음수/문자 입력은 불가 상태로 제한됨.
  - 단위(cc, km)는 고정 텍스트로 노출됨.
  - ‘1인 소유 여부’는 토글로 구성되며, OFF/ON 값이 차량
  - 신뢰도·감가 협의의 참고 지표로 프로필 요약 데이터에
  - 포함됨.
  - ‘사고/수리 이력’은 무사고 / 단순교환 있음 / 사고 이력
  - 있음 라디오로 구성되며, 선택값은 거래의 리스크 라벨링
  - 기준값으로 확정됨.
  - 선택값이 없는 상태에서는 다음 단계 진행이 제한됨.

### 2. 옵션 및 추가 정보
  - ‘차량 옵션 선택’은 다중 선택 가능한 칩(태그) 목록으로
  - 구성되며, 선택된 옵션만 활성 스타일로 유지됨.
  - 동일 옵션 중복 선택은 불가.
  - 옵션 선택 결과는 입찰 판단에 직접 사용되는 요약 정보
  - 로 입찰 카드/상세에 노출됨.
  - ‘특이사항/셀링 포인트’는 멀티라인 텍스트 입력으로 구
  - 성되며, 판매자가 차량 상태·강점·주의사항을 서술하는
  - 값으로 저장됨.
  - 미입력 시에도 거래 생성은 가능하나, 이후 딜러 판단에
  - 불리할 수 있음을 전제한 선택 입력값으로 처리됨.

### 3. 차량 사진 업로드
  - ‘사진 추가하기’는 이미지 업로드 슬롯으로 구성되며, 업
  - 로드된 사진은 썸네일로 리스트 노출됨.
  - 사진은 최대 20장까지 업로드 가능하며, 업로드 가능 형
  - 식(JPG/JPEG/PNG)과 가이드(해상도/용량 제한)가 안
  - 내 문구로 고정 노출됨.
  - 각 사진 썸네일에는 삭제(×) UI가 포함되며, 삭제 시 해당
  - 사진은 목록에서 즉시 제거됨.
  - 업로드 순서는 썸네일 배열 순서로 유지되며, 대표이미지
  - 는 첫 이미지 기준으로 반영됨

### 4. 입찰 조건 설정
  - ‘희망 낙찰가(참고용)’는 판매자 희망값 입력 영역이며,
  - 시스템이 딜러 입찰을 강제하는 값이 아니라 가격 기대치
  - 안내/참고 값으로 저장됨.
  - ‘입찰 기간’은 시작일~종료일 선택형 입력으로 구성되며,
  - 시작일이 현재 시점 이전으로 설정되는 경우는 제한됨.
  - 종료일은 시작일보다 빠르게 설정 불가이며, 위반 시 다
  - 음 단계 진행이 제한됨.
  - ‘입찰 시작가’는 입찰의 기준가로 저장되며, 이후 딜러 입
  - 찰가 유효성 검증의 기준값으로 사용됨(시작가 미입력 또
  - 는 0 이하 입력 시 제한됨).
  - ‘최소 호가 단위’는 드롭다운으로 구성되며, 딜러 입찰 금
  - 액이 해당 단위의 배수로만 인정되도록 입찰 유효성 기준
  - 값으로 확정됨.
  - ‘딜러 노출 국가(수출 가능 시)’는 수출 가능 선택 상태에
  - 서만 노출되며, 다중 선택 칩 형태로 입력됨(예: 홍콩, 일
  - 본).
  - 미선택 시 수출 입찰 참여 범위가 확정되지 않으므로 다
  - 음 단계 진행이 제한됨.

### 5. 등록 약관 동의
  - 약관 동의는 체크박스 리스트로 구성되며, 필수 동의 항
  - 목은 라벨로 구분됨.
  - 필수 동의 미체크 상태에서는 ‘다음’ 버튼 비활성화 처리

### 6. 취소
  - 클릭 시 본 화면에서 입력된 값은 확정되지 않으며, 등록
  - 플로우가 종료됨(직전 화면으로 이동).

### 7. 다음
  - 클릭 시 본 화면 전체 입력값에 대해 필수값 누락/형식 오
  - 류/정합성 오류(기간 역전, 수출 국가 미선택 등)가 검증
  - 됨.
  - 검증 통과 시 ‘등록내용확인’ 화면(FRT_034)으로 전환되
  - 며, 본 화면 입력값이 확인 화면에 그대로 바인딩되어 노
  - 출됨.
  - 검증 실패 시 화면 전환은 발생하지 않으며, 오류 항목은
  - 사용자가 즉시 수정 가능하도록 오류 상태로 강조 노출
  - 됨.
