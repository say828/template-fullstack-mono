# FRT_001 UI Spec

- screen code: `FRT_001`
- screen name: `랜딩페이지`
- primary route: `/`
- access: `public`
- source spec: `sdd/01_planning/02_screen/ir/seller/seller_screen_spec.pdf`
- page number: `2`

## Screen Requirements

### 1. 상단 로그인 버튼 영역
  - 비로그인 상태 기본 노출
  - 클릭 시 로그인 화면(FRT_002)으로 이동
  - 로그인 완료 시, 계정 유형(판매자/딜러)에 따라 해당 서
  - 비스 홈으로 전환

### 2. 회원가입 버튼
  - 클릭 시 역할 선택 화면(FRT_005) 및 회원가입 플로우
  - 진입
  - 이후 판매자/딜러 선택 값에 따라 각 서비스 회원가입 절
  - 차 분기

### 3. 메인 히어로 영역
  - PALKA 서비스의 핵심 가치 및 글로벌 경매 구조 요약 노
  - 출
  - 내 차량 판매하기 → 판매자 로그인 플로우 진입
  - (FRT_002)
  - 딜러로 입찰하기 → 딜러 로그인 플로우 진입
  - (FRT_003)
  - 모든 CTA는 비로그인 상태에서만 노출

### 4. 역할 선택 섹션
  - “판매자로 이용하기 / 딜러로 이용하기” 카드형 UI 구성
  - 각 카드 클릭 시 해당 역할 기준 회원가입 페이지로 이동
  - 각 역할별 제공 서비스 요약 문구 노출

### 5. 서비스 가치 안내 섹션
  - PALKA가 제공하는 핵심 서비스 항목 나열
  - 글로벌 딜러 네트워크
  - 차량 자동 조회
  - 전문 검차 서비스
  - 해외송금 및 수출 지원
  - 투명한 거래 절차

### 6. 해외 딜러 수출 프로세스 안내
  - 거래 진행 절차 7단계 시각화
  - 매물 입찰
  - PALKA 검차 평가
  - 판매자 감가 협의
  - 해외딜러 → PALKA 송금
  - PALKA → 판매자 정산
  - 수출 서류 처리
  - 도착 후 차량 인도

### 7. 푸터 영역
  - 서비스 운영 정보 및 약관 관련 링크를 제공하는 영역
  - 회사 정보 / 약관 / 개인정보처리방침 / 고객센터 정보 노
  - 출
  - SNS 링크 제공
