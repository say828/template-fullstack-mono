# FRT_008 UI Spec

- screen code: `FRT_008`
- screen name: `딜러회원가입`
- primary route: `/signup/dealer`
- access: `public`
- source spec: `sdd/01_planning/02_screen/ir/seller/seller_screen_spec.pdf`
- page number: `9`

## Screen Requirements

### 1. 기본 정보 입력 영역
  - 딜러 계정의 기본 인증 정보 및 연락 수단을 등록하는 영
  - 역.
  - 이름: 실명 입력, 필수
  - 휴대전화번호: 숫자만 허용, 거래 알림·승인 알림 기본 수
  - 신 수단
  - 이메일 주소: 로그인 계정 ID, 중복 불가
  - 비밀번호 / 비밀번호 확인: 보안 기준 충족 및 일치 필수
  - 활동 국가: 거래 통화·수출 정책·서비스 노출 범위 기준으
  - 로 사용

### 2. 사업자 정보 입력 영역
  - 딜러의 법적 사업자 신원 확인을 위한 필수 정보 영역.
  - 업체명: 사업자등록증 기준 입력
  - 사업자등록번호: 숫자만 허용, 중복 등록 불가
  - 사업자등록증 업로드: 필수 첨부 (PDF/JPG/PNG)
  - 사업장 주소: 실제 영업지 주소 입력 및 주소 검색 지원

### 3. 딜러 자격 정보 입력 영역
  - 딜러로서 활동 가능한 자격을 검증하기 위한 증빙 제출
  - 영역.
  - 매매사원증 번호 입력
  - 매매사원증 사본 업로드: 필수
  - 신분증 사본 업로드: 필수
  - 모든 파일은 관리자 승인 심사 기준 자료로 저장됨

### 4. 약관 동의 영역
  - 서비스 이용약관: 필수
  - 개인정보처리방침: 필수
  - 마케팅 정보 수신 동의: 선택
  - 필수 약관 미동의 시 가입 불가

### 5. 회원가입 완료 버튼
  - 모든 필수 입력 및 필수 서류 업로드 완료 시 활성화
  - 클릭 시:
  - 딜러 계정 생성
  - 계정 상태 → 승인 대기
  - 관리자 승인 검토 목록에 즉시 반영
  - 딜러 회원가입 완료 화면(FRT_009)으로 즉시 전환
  - 승인 완료 시 이메일 안내 발송

### 6. 로그인 이동
  - 기존 계정 보유 사용자를 위한 로그인 화면(FRT_003) 이
  - 동
