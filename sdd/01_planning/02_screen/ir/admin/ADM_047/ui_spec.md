# ADM_047 UI Spec

- screen code: `ADM_047`
- screen name: `리포트`
- primary route: `/admin/reports`
- access: `role:ADMIN`
- source spec: `sdd/01_planning/02_screen/ir/admin/admin_screen_spec.pdf`
- page number: `48`

## Screen Requirements

### 1. 기준 시점/갱신 정보 영역
  - 리포트 산출 기준 시점 고정 노출(예: 2025.05.21 14:12
  - 기준), 본 화면의 모든 지표·차트·랭킹의 기준 시점으로
  - 동일 적용.
  - 새로고침 동작 시 기준 시점이 최신으로 갱신되며, 화면
  - 내 모든 구성요소가 동일 기준으로 재계산되어 동기화됨.
  - 기준 시점은 “조회 시점”이 아닌 “집계 완료 시점” 의미로
  - 사용

### 2. 핵심 KPI 요약 카드 영역
  - 운영 핵심 지표 4종을 단일 기준으로 요약 노출하여 “현
  - 재 상태가 정상 범위인지”를 즉시 판별하는 영역.
  - 각 카드 공통 규칙
  - 주요 값(절대값) + 증감(직전 대비) 동시 노출.
  - 증감은 상승/하락 시각 구분(색상/아이콘) 고정.
  - 증감 기준은 “직전 기준 시점 대비”
  - 카드별 의미
  - 총 거래 완료 수: 완료 처리된 거래 건수 누적치(거래
  - 완료 정의는 거래 상태 체계의 최종 완료 단계로 고
  - 정).
  - 평균 감가율: 감가 협의가 발생한 거래의 평균 감가율
  - (감가율 산식 기준 고정).
  - 평균 거래 소요시간: 거래 생성/접수부터 완료까지 평
  - 균 소요(거래 단계 전환 시간 누적 기반).
  - 총 거래액(GMV): 거래 완료 기준의 총 거래금액 합계
  - (집계 포함/제외 조건 고정 필요: 취소/무효/반려 등
  - 제외).

### 3. 감가율 분포 차트 영역
  - 감가 품질을 “평균”이 아닌 “분포”로 판단하는 영역.
  - X축은 감가율 구간(예: 0%, 1~3%, 3~5%, 5~8%,
  - 8~10%, 10% 이상 등)으로 고정, Y축은 해당 구간의 거
  - 래 건수 집계.
  - 막대 hover/선택 시 해당 구간의 건수(count) 노출되어,
  - 특정 구간 급증 여부를 운영자가 즉시 확인 가능.

### 4. 거래 단계별 전환 및 이탈 퍼널 영역
  - 거래가 단계별로 얼마나 다음 단계로 전환되고 어디서 이
  - 탈되는지 시각화하는 병목 탐지 영역.
  - 퍼널 각 레벨은 거래 단계(예: 검차/감가/인도/정산 등 서
  - 비스 단계 체계 기준)로 고정 매핑.
  - 각 단계별 “잔여 거래 수(예: 전환대기 수)”가 노출되어,
  - 특정 단계 체류 급증 시 운영 대응(담당자 증원/정책 점
  - 검/알림 강화) 판단 근거로 사용.
  - “이탈” 정의는 상태 체계 내 종료/중단/반려/취소 등으로
  - 고정 필요(단순 미진행과 구분).

### 5. 단계별 평균 소요시간 바 차트 영역
  - 거래 단계별 평균 체류 시간을 비교하여 병목 단계를 정
  - 량화하는 영역.
  - 단계 목록은 서비스 단계 체계와 동일하게 고정(단계 추
  - 가/삭제 시 리포트 항목도 동기화).
  - 각 바는 평균 소요시간(일 단위 등)으로 표기되며, hover
  - 시 단계명+정확 소요시간 노출.
  - “소요시간” 기준점은 단계 진입 시점부터 다음 단계 확정
  - 시점까지로 고정

### 6. 우수/관리 딜러 랭킹 테이블 영역
  - 거래 성과와 품질을 기반으로 딜러를 상대 비교하는 운영
  - 용 랭킹 영역.
  - 테이블은 순위 기반 정렬 고정(기본값: 거래 완료율).
  - 컬럼 의미
  - 순위/딜러/업체명: 식별 정보.
  - 낙찰 성공: 낙찰 확정 건수.
  - 거래 완료율: 참여 거래 대비 완료 거래 비율(분모 정
  - 의 고정 필요).
  - 평균 감가율: 해당 딜러 거래의 평균 감가율(산식 동
  - 일 적용).
  - 총 매입액: 완료 거래 기준 총 금액 합.
  - 완료율은 진행바로 시각화되어, 고성과/저성과를 즉시
  - 구분 가능
