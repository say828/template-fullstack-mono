## Screen Requirements

### 1. 대시보드

### 3. 아코디언 기능
  - 모든 메뉴는 기본적으로 접힘(collapsed) 상태로 로드
  - 메뉴 그룹을 클릭하면 하위 서브메뉴가 펼쳐짐
  - 다시 클릭하면 접힘
  - 접기/펼치기 상태는 로컬 저장소에 저장
  - 선택된 메뉴 강조(Active) 상태 표시
  - 각 메뉴 항목은 화면 ID와 매핑되어 클릭 시 연결된 화면
  - 호출
  - 사용자 권한에 따라 일부 메뉴 비활성화
