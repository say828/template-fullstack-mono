## Screen Requirements

### 1. FAQ 콘텐츠 입력 영역
  - FAQ 단일 콘텐츠의 정책 단위 입력 영역.
  - 채널 : 콘텐츠 노출 대상 서비스 범위 결정 기준값.
  - - 판매자 / 딜러 중 단일 선택 구조.
  - - 선택 값에 따라 해당 서비스 FAQ 목록에만 노출 적용.
  - 카테고리 : 서비스 기능 분류 기준값. 동일 채널 내 콘텐츠
  - 정렬·탐색·필터 기준으로 사용됨.
  - 질문 제목 : 사용자 FAQ 목록에 노출되는 메인 질문 문구.
  - 검색·목록 정렬·클릭 진입 기준 키값.
  - 답변 내용 : FAQ 본문 콘텐츠 영역. 서식·링크·이미지·첨
  - 부 자료 삽입 가능. 사용자 화면에서 그대로 노출되는 최
  - 종 콘텐츠 데이터.
  - 노출 여부 : 서비스 화면 실제 노출 제어 스위치. 비노출
  - 상태의 콘텐츠는 관리 화면에만 유지되며 사용자 서비스
  - 화면에는 미노출.

### 2. 취소 / 삭제 버튼 영역
  - 진입 유형에 따른 제어 분기 영역.
  - 취소 버튼 클릭 시 팝업 닫기.
  - 등록 진입 시 : 버튼 구성(취소 / 저장)
  - 수정 진입 시 : 버튼 구성(삭제 / 저장)
  - 삭제 선택 시 해당 FAQ 콘텐츠는 즉시 목록에서 제거되
  - 며 사용자 화면에서도 동시 제거

### 3. 저장 버튼
  - 입력된 FAQ 콘텐츠를 정책 데이터로 확정 반영하는 트리
  - 거.
  - 채널·카테고리·노출여부·질문·답변 데이터 일괄 저장.
  - 저장 완료 즉시 FAQ 목록 반영.
  - 노출 상태일 경우 서비스 화면 동기 반영.
