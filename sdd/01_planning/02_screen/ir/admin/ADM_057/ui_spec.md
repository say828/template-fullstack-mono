- primary route: `/admin/support/inquiries/:inquiryId`
- access: `role:ADMIN`
- source spec: `sdd/01_planning/02_screen/ir/admin/admin_screen_spec.pdf`
- page number: `58`

## Screen Requirements

### 1. 문의 메타 요약 영역
  - 목록에서 선택된 문의의 기준 정보 고정 노출.
  - 이메일: 회신 수신 대상 주소(답변 완료 시 발송 대상)
  - 문의 채널: 판매자/딜러 구분 값(문의 주체 확정 기준)
  - 문의 유형: 문의 분류 값
  - 접수일: 문의 접수 시각(정렬/기간 필터 기준과 동일
  - 의미)
  - 담당자: 해당 문의 처리 담당 운영자 식별 값
  - 상태 태그: 접수/답변완료 시각 구분

### 2. 문의 정보(원문) 영역
  - 문의자가 입력한 원문 데이터 열람 구간.
  - 제목/내용은 수정 불가 상태로 고정 노출.
  - 첨부 이미지(또는 파일) 존재 시 썸네일 형태로 노출
  - 첨부가 없는 경우: 첨부 영역 미노출
  - 본 영역 데이터는 답변 내용과 무관하게 “문의 접수 시점
  - 값”으로 고정.

### 3. 답변 작성/확정 영역
  - 운영자가 문의에 대한 답변을 입력하고 “답변완료”로 확
  - 정하는 처리 구간.
  - 상태 = 접수
  - 답변 입력 가능 상태.
  - 답변 완료 버튼 노출.
  - 버튼 클릭 시 “답변 확정” 단계로 진행(처리 결과:
  - 상태 전환 + 회신 발송 + 처리일시 기록).
  - 상태 = 답변완료
  - 답변 입력 영역: 기존 답변 내용 읽기전용으로 고
  - 정 노출.
  - 답변 완료 버튼 미노출.
