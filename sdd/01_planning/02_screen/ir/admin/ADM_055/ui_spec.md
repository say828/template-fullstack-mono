# ADM_055 UI Spec

- screen code: `ADM_055`
- screen name: `버전보기`
- primary route: `/admin/version`
- access: `role:ADMIN`
- source spec: `sdd/01_planning/02_screen/ir/admin/admin_screen_spec.pdf`
- page number: `56`

## Screen Requirements

### 1. 버전 이력 목록 영역
  - 선택된 정책 문서의 전체 개정 이력을 시간순으로 제공하
  - 는 기준 테이블.
  - 버전
  - 해당 정책 문서의 개정 식별 값.
  - 상위 버전일수록 최신 문서.
  - 정책 분쟁·운영 판단의 기준 식별자.
  - 수정일
  - 해당 버전이 최종 확정 저장된 날짜.
  - 서비스 운영 기준 효력 발생 시점 추적 기준 값.
  - 수정자
  - 해당 버전을 저장 확정한 관리자 계정 식별자.
  - 정책 책임자 추적용 운영 로그 성격 데이터.
  - 노출 여부
  - 사용자 화면 적용 상태.
  - 노출: 현재 사용자 서비스에서 효력 적용 중인 기준
  - 문서
  - 비노출: 내부 참고·과거 이력 문서
  - 내용보기
  - 버전별 상세 확인을 위한 내용보기 버튼 제공

### 2. 내용보기 버튼
  - 선택한 정책 문서 버전의 상세 내용을 확인하기 위한 진
  - 입 버튼
  - 버전 상태에 따라 화면 전환 방식이 달라짐
  - 현재 노출 중인 최신 버전
  - 내용보기 클릭 시 ADM_054 (정책 문서 등록/수정
  - 화면) 으로 이동
  - 모든 입력 필드 활성화 상태
  - 수정 및 저장 가능
  - 과거 버전 또는 비노출 버전
  - 내용보기 클릭 시 ADM_054 화면으로 이동하되, 모
  - 든 입력 항목이 비활성화된 읽기 전용 상태로 노출
  - 저장, 수정, 노출 변경 불가
  - 과거 버전은 이력 조회 목적만 허용되며, 직접 수정 불가

### 3. 확인
  - 클릭 시 버전 이력 조회를 종료하고 이전 화면(정책 문서
  - 관리 목록)으로 복귀
