# ADM_059 UI Spec

- screen code: `ADM_059`
- screen name: `계정등록상세`
- primary route: `/admin/accounts/:accountId`
- access: `role:ADMIN`
- source spec: `sdd/01_planning/02_screen/ir/admin/admin_screen_spec.pdf`
- page number: `60`

## Screen Requirements

### 1. 관리자 기본 정보 입력 영역
  - 관리자 ID
  - 관리자 계정 생성 시 자동 생성된 식별자 값으로 노출
  - 됨
  - 수정 불가 상태로 표시되며, 시스템 내 고유 값으로 사
  - 용됨
  - 이름
  - 관리자 식별을 위한 실명 또는 운영용 이름 입력 필드
  - 목록, 로그, 변경 이력 등에서 관리자 식별 기준으로
  - 사용됨
  - 이메일
  - 관리자 로그인 및 운영 알림 수신 기준 이메일
  - 중복 이메일 등록 불가
  - 연락처
  - 운영 이슈 발생 시 관리자에게 직접 연락 가능한 정보
  - 로 사용됨
  - 권한 그룹
  - 관리자 계정에 부여할 접근 권한 범위를 선택
  - 선택된 권한 그룹에 따라 접근 가능한 메뉴 및 기능
  - 범위가 결정됨
  - 계정 상태
  - 사용 : 계정 생성 즉시 관리자 시스템 로그인 및 이용
  - 가능 상태
  - 사용중지 : 계정은 생성되나, 관리자 시스템 접근은 차
  - 단된 상태
  - 초기 기본값은 사용으로 설정됨

### 2. 취소
  - 입력 중인 관리자 계정 등록 절차를 중단하고 이전 화면
  - 으로 복귀
  - 입력된 모든 정보는 저장되지 않음

### 3. 저장
  - 입력된 관리자 계정 정보를 기준으로 신규 관리자 계정
  - 생성 처리
  - 필수 항목 유효성 검증 통과 시에만 등록 가능
  - 등록 완료 후 관리자 계정 목록 화면으로 전환됨
