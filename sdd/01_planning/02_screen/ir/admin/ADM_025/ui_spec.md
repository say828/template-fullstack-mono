## Screen Requirements

### 1. 감가 협의 상태/필터 영역
  - 감가 협의 레코드를 상태 기준으로 조회 범위를 확정하는
  - 제어 영역.
  - 상태 탭은 감가 협의 레코드의 진행 상태로 필터링됨
  - - 전체: 감가 협의 레코드 존재 여부와 무관하게 감가
  - 협의 대상 거래 전체
  - - 제안 필요: 검차완료 상태이며 딜러 감가 제안 레코
  - 드가 미생성 상태
  - - 승인대기: 딜러 감가 제안이 존재하며 판매자 최종
  - 응답(승인/재협의 요청)이 미확정 상태
  - - 재협의: 판매자 재협의 요청 이후 딜러 재제안 또는
  - 추가 합의 진행 중 상태
  - - 최종확정: 판매자 승인으로 감가 금액이 확정되어 이
  - 후 인도/정산 기준 금액으로 고정된 상태
  - - 지연: 감가 협의 SLA가 정책 임계치를 초과했거나
  - 지연 플래그가 부여된 상태(상태값 자체가 아닌 운영용
  - 분류)
  - 기간 프리셋은 최종 업데이트 일시 기준으로 조회 범위가
  - 결정됨
  - - 오늘/최근 7일/최근 30일/전체는 서버 조회 파라미
  - 터로 고정 전달됨
  - 검색은 목록의 식별 키를 단일 입력으로 조회 범위를 추
  - 가 축소함
  - - 검색 키: 거래ID, 차량번호, 판매자명, 딜러명(복수 매
  - 칭)
  - - 입력값은 공백 포함 문자열로 전달되며 부분 일치 기
  - 준으로 필터링됨
  - 초기화는 상태/기간/검색 조건을 초기 기본값(상태=전
  - 체, 기간=전체, 검색=빈값)으로 복원함

### 2. 상태별 대기 건수 요약 카드 영역
  - 현재 조회 조건(기간/검색 적용 전역 기준)에서 감가 협
  - 의 병목을 상태 단위로 수치화하여 노출하는 영역.
  - 각 카드의 건수는 거래 단위가 아니라 감가 협의 레코드
  - 기준으로 집계됨
  - - 제안 필요는 감가 협의 레코드 미생성 거래를 대상으
  - 로 별도 집계됨
  - - 승인대기/재협의/최종확정은 감가 협의 레코드의 최
  - 신 상태값 기준으로 집계됨
  - - 지연은 감가 협의 상태와 무관하게 SLA/지연 플래그
  - 조건으로 교차 집계됨
  - 카드 클릭 시 해당 상태 탭과 동일한 필터가 즉시 적용되
  - 어 목록이 재조회됨
  - 집계 기준 시점은 화면 상단 “기준 시각”과 동일하며 재조
  - 회 시 동기 갱신됨

### 3. 감가 협의 거래 목록 영역
  - 감가 협의 진행 현황을 거래별로 비교·판단하기 위한 목
  - 록 영역. 각 행은 감가 협의의 “현재 합의 수준”과 “금액
  - 변화량”을 한 화면에서 판단 가능하도록 구성됨.
  - 목록 정렬 기준은 최종 업데이트 내림차순(최신 우선)으
  - 로 고정됨(운영 우선 순위 기준)
  - • 거래ID는 거래 단위 고유 식별자
  - 차량 정보: 거래 생성 시 확정된 차량 기본 정보 요약(모
  - 델/연식/주행거리 등), 감가 판단의 비교 기준 역할
  - 판매자 / 딜러: 감가 협의 당사자 식별 정보, 운영 담당자
  - 커뮤니케이션 기준 키
  - 상태: 감가 협의 최신 상태값, 상태 탭 필터의 기준값과
  - 1:1 매칭됨
  - 최초 금액: 감가 협의 시작 기준 금액
  - - 검차/감가 이전 단계에서 확정된 낙찰 금액 또는 거
  - 래 기본 금액이 원천값으로 사용됨
  - - 최종확정 이후에도 최초 금액은 변경되지 않으며 비
  - 교 기준으로만 유지됨
  - 제안 금액 변화: 최초 금액 대비 현재 제안(또는 합의) 금
  - 액의 증감값
  - - 산식: 현재 금액(최신 제안 또는 최종확정 금액) - 최
  - 초 금액
  - - 음수는 감가(차감) 방향, 양수는 예외적으로 증액이
  - 발생한 경우를 표현 가능
  - 최종 금액: 현재 단계에서 유효한 기준 금액
  - - 승인대기/재협의: 최신 딜러 제안 금액이 노출됨(판
  - 매자 확정 전이라도 운영 판단 기준으로 사용)
  - - 최종확정: 판매자 승인으로 확정된 감가 반영 금액이
  - 고정 노출됨
  - - 제안 필요: 제안 레코드가 없으므로 공란 또는 “-”로
  - 유지됨
  - SLA: 감가 협의 단계 경과일 또는 마감까지 남은 시간을
  - D- / D+ 형태로 표기한 운영 지표
  - - 기준 시점은 감가 협의 시작(첫 제안 등록) 또는 검차
  - 완료 전환 시점 중 정책 기준으로 결정됨
  - - 임계치 초과 시 지연 분류(지연 탭/카드)에 포함될
  - 수 있음
  - 최종 업데이트: 감가 협의와 관련된 최신 이벤트 발생 시
  - 각
  - - 제안 등록/재제안/판매자 승인/재협의 요청 등 상태
  - 변화 이벤트 기준으로 갱신됨
  - 리스트행 클릭 시 해당하는 거래의 감가 협의 상세
  - (ADM_026 등) 화면으로 이동됨
