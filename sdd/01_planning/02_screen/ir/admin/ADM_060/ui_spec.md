# ADM_060 UI Spec

- screen code: `ADM_060`
- screen name: `관리자권한관리`
- primary route: `/admin/permissions`
- access: `role:ADMIN`
- source spec: `sdd/01_planning/02_screen/ir/admin/admin_screen_spec.pdf`
- page number: `61`

## Screen Requirements

### 1. 권한 그룹 추가
  - 신규 관리자 권한 그룹 생성을 위한 진입 버튼
  - 클릭 시 권한 그룹 추가 팝업(ADM_061)을 호출함

### 2. 권한 그룹 목록
  - 현재 시스템에 등록된 관리자 권한 그룹 목록
  - 각 항목은 하나의 관리자 권한 정책 단위로 동작함
  - 권한 그룹 선택 시 우측 권한 설정 영역이 해당 그룹 기준
  - 으로 갱신됨

### 3. 권한 그룹 수정
  - 클릭 시 선택된 권한 그룹의 수정(ADM_061) 화면을 호
  - 출함
  - 권한 그룹명, 설명 등 메타 정보 수정 용도로 사용됨
  - 권한 자체 변경은 우측 권한 설정 영역에서 수행됨

### 4. 권한 그룹 삭제
  - 선택된 권한 그룹 삭제 기능
  - 해당 권한 그룹이 어느 관리자 계정에도 할당되어 있지
  - 않은 경우에만 삭제 가능
  - 사용 중인 권한 그룹은 삭제 불가 상태로 유지됨

### 5. 권한 설정 영역
  - 선택된 권한 그룹에 대한 메뉴 단위 권한 설정 영역
  - 각 행은 하나의 관리자 메뉴 단위를 의미함
  - 조회 권한: 메뉴 접근 및 정보 조회 가능 여부
  - 등록/수정 권한: 데이터 생성 및 수정 가능 여부
  - 삭제 권한: 데이터 삭제 가능 여부
  - 설정 권한: 정책·설정 변경 가능 여부
  - 체크 상태에 따라 해당 권한 그룹의 접근 범위가 정의됨

### 6. 저장
  - 현재 선택된 권한 그룹의 권한 설정 저장
  - 저장 시 해당 권한 그룹에 소속된 관리자 계정에 변경된
  - 권한 정책이 일괄 적용됨
  - 저장 전까지 변경 사항은 임시 상태로 유지됨
