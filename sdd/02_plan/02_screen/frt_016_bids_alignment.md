# FRT_016 입찰현황 정합성 정리

- 화면 코드: `FRT_016`
- route: `/seller/vehicles/:vehicleId/bids`
- 상태: `implemented`

## 반영 항목

- 차량 요약 영역을 실시간 입찰 데이터 기준 노출로 정리(현재 최고 입찰가/입찰 건수/마감기한/남은시간/이미지).
- 우측 상단 고정 `차량 상세 보기` CTA 추가 및 링크 유지.
- 입찰 목록을 카드형에서 테이블형(`딜러명/입찰가/입찰 시간/국가/비고`)으로 정렬.
- 기본 정렬을 입찰가 내림차순으로 유지하고 최고가 행에 `Highest` 표시 강화.
- 더 보기 + 버튼에서 추가 조회 로드 시 스크롤 위치 유지.
- 입찰 관리 패널의 남은시간 카운트다운 및 두 토글(새 입찰 알림 받기/최고가 자동 갱신 알림)을 계정 설정 기준으로 저장되도록 연결.
